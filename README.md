## 文本轉換工具-LINE
1. 下載**文本轉換工具-LINE.exe**
2. 選擇文本格式是「私訊」或「群組」
3. 輸入主視角的名稱（可以不填
4. 左方輸入框複製貼上文本
   - 第一行必須是聊天對象／群組
   - 角色名稱後須使用全形冒號
   - 訊息內容、時間戳、讀取回條之間須以半形空白分隔
6. 點擊「轉換」
7. 右方輸入框出現添加HTML標籤的文本
8. 點擊「複製」
9. 貼在AO3的HTML編輯器裡，完成～
